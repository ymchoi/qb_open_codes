# Max Mode

`/max`와 함께 입력된 프롬프트를 수행할 때 아래 지침을 적용한다.

## 최우선 원칙

완벽함 / 철저함 / 정확함 / 논리적 / 사실기반 / 실수없음이 최우선이다. 수행의 효율성이나 작업시간은 후순위다.

## 수행 절차

1. **CLAUDE.md 재확인**: 프로젝트 루트의 CLAUDE.md를 지금 다시 읽고, 그 안의 모든 지침을 준수한다.

2. **ultrathink**: 작업에 착수하기 전에 좀 더 ultrathink 한다. 논리와 팩트를 기반으로 접근 방식을 설계한다. 추측으로 진행하지 않는다.

3. **정보 수집**: 로컬(프로젝트 파일, 코드, 설정)과 웹(docs MCP, 웹검색)에서 관련 정보를 충분히 수집한다. 수집이 충분하다고 확신할 때까지 작업에 착수하지 않는다.

4. **수집 결과 기반 수행**: 수집한 정보를 근거로 작업을 수행한다. 근거 없는 판단을 하지 않는다.

5. **검증**: 결과물이 논리적으로 정확하고 사실에 부합하는지 확인한 뒤 완료한다.
